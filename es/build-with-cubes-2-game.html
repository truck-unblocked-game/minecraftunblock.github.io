Instrucciones del juego

Llaves de flecha / wasd para moverse y controlar el vuelo;Sostenga LMB y mueva el mouse para rotar la cámara;Barra espaciadora - para saltar;E para recoger objetos;LMB Haga clic para seleccionar bloque y colocar uno nuevo;RMB para destruir cualquier bloque y el objeto elegido;Z y Q - Crear portales

¡Obtenga una nueva experiencia en la construcción de bloques 3D!Build with Cubes 2 es una caja de arena de estilo Voxel estilo Minecraft donde puede construir lo que desee con los bloques de cubos. Use un plan simple lleno de cubos o genere un paisaje 3D utilizando configuraciones como semillas, altura y ancho de las colinas, ajuste el número de objetos, entra, árboles, flores.Construya una casa de muñecas, su propia casa o castillo, o incluso un pueblo pequeño, proporcione a la casa muebles, organice una variedad de decoración y elementos ligeros. Use portales, superficies portales y el portalgu para crear juegos de mini rompecabezas, teletransportes, edificios preparados, etc.Llevando su mundo con entidades y animales, cree melodías simples usando bloques de notas musicales, use el modo de mosca y las funciones de espejo para construir más rápido y más conveniente.Experimente con los colegas de games de física como bolas, dados hinchables, trampolines y TNT.
